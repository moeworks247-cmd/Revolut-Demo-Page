# AI Trading Copilot · Product Portfolio

A premium fintech portfolio website showcasing **AI Trading Copilot** — a conceptual AI-powered investing assistant designed for Revolut-style retail trading platforms.

## What's Included

### Portfolio Website (9 Sections)
1. **Hero** — Cinematic intro with animated AI dashboard mockup
2. **The Problem** — Retail investing pain points with data visualisations
3. **The Solution** — Six AI-powered feature cards
4. **Interactive Prototype** — Horizontal scroll iPhone 15 Pro mockups (6 screens)
5. **Architecture** — Animated systems diagram (User → API → AI → Risk → Notifications)
6. **Roadmap** — Interactive Q1–Q4 product strategy timeline
7. **KPI & Impact** — Animated metrics, North Star metric, projection charts
8. **Why Revolut** — Strategic positioning and differentiation
9. **Final CTA** — Cinematic closing section

### Mobile Prototype Screens
- Portfolio Dashboard
- Stock Detail (NVDA)
- AI Risk Warning Modal
- Market Insights Feed
- Portfolio Health Dashboard
- Beginner Mode Settings

## Run Locally

```bash
npm install
npm run dev
```

Open the URL shown in terminal (typically `http://localhost:5173`).

## Tech Stack

- React 18 + Vite
- Tailwind CSS (dark mode, glassmorphism, gradients)
- Framer Motion (scroll reveals, micro-interactions, loading screen)
- Recharts (charts and data visualisations)
- Lucide React (iconography)
- Instrument Serif + Inter (editorial typography)

## Design

- Investor-ready, Series A fintech startup aesthetic
- Sticky navigation with smooth scroll
- Premium loading transition
- Responsive mobile-first layout
