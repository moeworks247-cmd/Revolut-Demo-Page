# REVA · Revolut Enhanced Value Advisor

A premium fintech portfolio website showcasing **REVA** (Revolut Enhanced Value Advisor), a conceptual AI powered investing assistant designed for Revolut style retail trading platforms.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide icons

## Project structure

- `src/components/website/` — Portfolio landing sections
- `src/components/screens/` — Mobile prototype screens
- `src/data/featureRoadmap.js` — Feature taxonomy and roadmap
- `public/hive/` — HIVE investor portrait assets
